3.Python数据类型：哈希/不可哈希

哈希类型。不可变类型，可利用hash函数查看其hash值，也可以作为字典的key。具体有：

数字类型：int, float, decimal.Decimal, fraction.Fraction, complex

字符串类型：str, bytes

元祖：tuple

冻结集合：frozenset

布尔类型：True， False

None

不可哈希类型。可变类型，不可以作为字典的key。有：list, dict, set
