测试角度学习 cache2go

Aug 20th 2019

cache2go 介绍

“Concurrency-safe golang caching library with expiration capabilities.”。作者如是介绍该项目，翻译后的意思为：存在心跳机制且并发安全的go语言缓存库。

测试角度学习 cache2go

我们知道心跳机制、并发安全是 cache2go 的两大特点，那么肯定是测试的重点了。

首先查看 cache_test.go ，可以获得以下信息：【 Cachetable 简称 table ， cacheitem 简称 item 】

| - | - | - |
| TestCache | 创建 table ，添加 item ，获取 item 基本信息功能验证 | Cache、table.Add、table.value、item.AccessCount、item.LifeSpan、item.AccessedOn、item.CreateOn |
| | | |
| TestCacheExpire | 心跳机制功能验证 | Cache、table.Add、table.value |
| TestExists | table 表中是否存在 item 项功能验证 | Cache、table.Add、table.Exists |
| TestNotFoundAdd | table 表中是否存在 item 项，若不存在则添加 功能验证 | Cache、table.NotFoundAdd |
